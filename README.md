## Node.js实战开发

> 这是一个开发中的项目，已实现部分接口(注册/登录/注销/查询用户详情/消息置顶/分页查询等...)，持续commit，敬请期待

#### 1. 技术栈
##### 1. Koa2
##### 2. sequelize and mysql（ORM框架 and 数据库）
##### 3. log4js（日志记录）
##### 4. jsonwebtoken （权限认证）


#### 2. 启动项目
##### 1. yarn / npm install（安装依赖）

##### 2. start mysql(安装、创建database：test)

##### 3. 开发环境：yarn/npm run dev; 生产环境： yarn/npm run prd;
